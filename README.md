I build edge-native apps and AI tools that solve real problems. Most of my stack runs on Cloudflare Workers and D1, wired up to Claude or LangGraph.js for the AI layer.

My public work skews toward prompt engineering, agentic research, and cultural AI. My private work covers dashboards, exam engines, ROI calculators, and team management tools.

## Featured Projects

### [IsItPolite](https://github.com/trbbmnglg/isitpolite)
AI cultural-etiquette checker for 195 countries. LangGraph.js agents (DeepSeek + Tavily) deliver grounded verdicts, country guides, quizzes, and community accuracy scoring — all on Cloudflare Pages + Functions + D1. Privacy-first, no personal data stored.

`LangGraph.js` `DeepSeek` `Tavily` `Cloudflare D1` `Agentic AI` `Programmatic SEO`

---

### [Prompt Lab](https://github.com/trbbmnglg/promptlab)
Interactive field guide to prompt engineering. Structured techniques, live examples, and export-to-image — rate-limited at the edge via Cloudflare Workers + KV. Built with React 18.

`React` `Cloudflare Workers` `KV` `Prompt Engineering` `LLM`

---

### [Research Agent CF](https://github.com/trbbmnglg/research-agent-cf)
Autonomous AI research agent running on Cloudflare. Calls search tools to gather targeted AI news, scores results against a personal relevance profile, strips hype, and self-critiques its draft before re-searching gaps. Learns from prior runs to cover new ground each cycle.

`TypeScript` `Agentic AI` `Claude` `Cloudflare Workers`

---

### [Guardrails Analysis](https://github.com/trbbmnglg/guardrailsanalysis)
Interactive analysis and visualization of AI guardrail systems. Deployed on Hugging Face Spaces, auto-synced from GitHub via Actions.

`AI Safety` `Guardrails` `Hugging Face Spaces` `Visualization`

---

### [Anya Bot](https://github.com/trbbmnglg/anya-bot)
COVID-19 info chatbot in Filipino (Tagalog), built with Python and the RASA framework. One of my earlier NLP experiments — still the repo with the most forks.

`Python` `RASA` `Conversational AI` `Tagalog NLP`
